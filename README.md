# README

Click is a basic photo storage application, built in Rails.

Getting Started:

To get started with the app, clone the repo then install the needed gems:

$ bundle install

Next, migrate the database:

$ rake db:migrate

Then run the app in a local server:

$ rails server

And navigate to:

localhost:3000
